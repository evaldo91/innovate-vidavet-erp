# Core ERP Veterinário & Agro

Sistema de gestão robusto desenvolvido para operações clínicas e rurais.

## 🏢 Clientes Ativos
- **Cliente:** Innovate / Vidavet
- **Status:** Fase de Implantação (Modelagem de Dados)
- **Perfil:** Clínica Veterinária e Atendimento a Fazendas

## 🚀 Funcionalidades do Produto
- Faturamento mensal consolidado para produtores rurais.
- Gestão de estoque com fator de conversão e ficha técnica.
- Módulo fiscal integrado (NFe/NFSe).
- Cadastro multi-propriedade com herança de endereços.

## 📁 Estrutura de Pastas
- `docs/database/`: Esquemas DBML do banco de dados.
- `sql/`: Scripts de migração e inicialização.
